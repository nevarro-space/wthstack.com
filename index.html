WTH Stack

Introducing a new desktop application stack for web developers who hate JavaScript!

What is WTH Stack?

There are three components to the WTH Stack:

  • Wails — a Go library for building cross-platform applications.
  • Templ — a templating language based on Go for HTML, CSS, and JavaScript.
  • HTMX — a small JavaScript library for declaratively adding interactivity to web pages in HTML.

What the Hell is This?

Good question! The WTH stack is for people who hate JavaScript, but still want to build HTML/CSS-based desktop applications.
